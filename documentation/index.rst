Welcome to Topaz3 documentation!
==================================

**Topaz3** is a tool for data transformation and machine learning, developed at `Diamond Light Source <https://www.diamond.ac.uk/>`_.

The code can be found in this repository: `<https://github.com/DiamondLightSource/python-topaz3/>`_

The documentation here is designed to provide clearer explanations of how the module provided work,
so that they can be adapted to different situations.
